Posture Detector

Team: Keaton, Gage, Roman, Brett
Wearable posture monitoring device with real-time feedback. Features TinyCore MCU with integrated accelerometer, Kalman Filter for signal processing, buzzer alert system for posture correction, and continuous tilt angle monitoring.

Key Features

  • TinyCore MCU with integrated accelerometer for motion sensing
  • Kalman Filter algorithm for accurate signal processing
  • Real-time tilt angle calculation and monitoring
  • Buzzer alert system for immediate posture correction feedback
  • Continuous posture tracking throughout the day
  • Wearable design for comfortable all-day use
  • Low-power operation for extended battery life

Thermohelm

Team: Roman Lynch, Brett Rabbiner, Gage Gruidel
Safety helmet with real-time temperature monitoring system. Features dual 10K thermistors for ambient and scalp temperature sensing, Arduino Nano 33 IOT microcontroller, Steinhart-Hart equation temperature calculations, and 105°F+ alert system with LED indicators.

Key Features

  • Dual 10K thermistors for ambient and scalp temperature monitoring
  • Arduino Nano 33 IOT microcontroller with WiFi capability
  • Steinhart-Hart equation for precise temperature calculations
  • 105°F+ threshold alert system for heat stress detection
  • LED indicators for visual temperature warnings
  • Real-time temperature data processing and display
  • Safety helmet integration for construction/industrial use

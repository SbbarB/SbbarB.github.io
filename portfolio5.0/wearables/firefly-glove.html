Firefly Glove (Soft Button)

Team: Roxy, Tessa, Kunstin, Brett
E-textile wearable with LED circuit activated by soft fabric button. Features zinc alloy button components, 220Ω resistor, conductive thread circuitry, and integrated LED illumination system.

Key Features

  • Soft fabric button mechanism with conductive pathways
  • Zinc alloy button components for durability
  • 220Ω current-limiting resistor for LED protection
  • Conductive thread circuitry integrated into glove fabric
  • LED illumination system activated by button press
  • Wearable e-textile design combining functionality with comfort

Wearable Focus Clapper

Team: Brett Rabbiner, Roman Lynch, Gage Gruidel
Film production wearable device for focus pulling workflow. Features laser-cut acrylic clapper mechanism, Arduino microcontroller, dual potentiometers for focus/iris control, 7-segment LED display, and push-button trigger system.

Key Features

  • Laser-cut acrylic clapper mechanism for precise actuation
  • Arduino microcontroller for real-time processing
  • Dual potentiometers for focus and iris control
  • 7-segment LED display for visual feedback
  • Push-button trigger system for clapper activation
  • Wearable design for hands-free operation during filming
  • Film production workflow integration
